# Tiki Temple Dude [ALPHA]

More like Icy Igloo Penguin, but hey... it's an alpha build.

## Controls

Player 1
 - Movement: WASD
 - Use Item: Spacebar

Player 2
 - Movement: Arrow Keys
 - Use Item: Enter

## Basic Features

Coins:
 - Affect player speed. Capped at 10.

Items:
 - Kick
 -- Opponent must be on the opposite lane for item to work.
 -- Removes coins from opponent.
